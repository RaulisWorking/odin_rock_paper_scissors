# odin_rock_paper_scissors
A 5-round Rock, Paper, Scissors game you play via browser prompt.
My first script written in JavaScript.

The game can only be played in DevTools Console

PLAY
- Choose rock, paper, or scissors when prompted.
- Computer picks randomly.
- After 5 rounds, the winner and final score show in the console

HIGHLIGHTS
- Input validation
- Random computer choice
- Round logic using if/else
- Score tracking + loop
